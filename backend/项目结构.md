project/
├── cmd/                     # 程序入口 (main.go 等)
├── config/                  # 配置文件（YAML/JSON/env）
├── internal/                # 内部实现逻辑
│   ├── controller/          # 控制器层，处理 HTTP 请求/响应
│   ├── service/             # 服务层，封装业务逻辑
│   ├── dao/                 # 数据访问层（SQL/Mongo/Redis）
│   │   ├── mysql/           # MySQL 相关数据访问
│   │   ├── mongo/           # MongoDB 相关数据访问
│   │   ├── redis/           # Redis 缓存操作
│   │   └── repository.go    # 统一对外的 DAO 接口
│   ├── storage/             # 对象存储 (MinIO 封装)
│   │   └── minio.go
│   ├── model/               # 数据模型（结构体定义，ORM/ODM）
│   │   ├── mysql/           # MySQL 对应的模型
│   │   ├── mongo/           # MongoDB 对应的模型
│   │   └── redis/           # Redis 使用的 key 定义
│   ├── middleware/          # Gin 中间件（认证、日志、限流）
│   ├── router/              # 路由注册
│   └── util/                # 工具类函数 (JWT、加密、日志封装等)
├── pkg/                     # 公共包（可复用，不依赖业务逻辑）
│   ├── logger/              # 日志封装
│   ├── database/            # 数据库连接初始化 (MySQL、Mongo、Redis)
│   └── storage/             # MinIO 客户端初始化
├── scripts/                 # 部署、启动脚本
├── test/                    # 单元测试
└── go.mod

| 场景   | 模板                     | 示例 key                            |
| ---- | ---------------------- | --------------------------------- |
| 缓存   | `cache:<业务>:<对象>:<id>` | `cache:user:profile:10086`        |
| 分布式锁 | `lock:<业务>:<资源>`       | `lock:order:20240902120001`       |
| 计数器  | `cnt:<业务>:<维度>:<日期>`   | `cnt:article:view:20240902`       |
| 限流   | `rl:<业务>:<用户>`         | `rl:login:ip:192.168.1.10`        |
| 消息队列 | `mq:<队列名>`             | `mq:email:register`               |
| 验证码  | `code:<场景>:<邮箱>`       | `code:register:alice@example.com` |
| 排行榜  | `rank:<业务>:<周期>`       | `rank:game:score:daily`           |
